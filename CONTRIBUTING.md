# Contributing

Thanks for your interest in contributing to the Astropods agents repo.

## How to contribute

1. **Fork** the repository
2. **Create a branch** for your changes
3. **Add or update an agent** — each agent lives in its own directory with an `astropods.yml` spec
4. **Open a pull request** with a description of your agent and how to run it

## Adding a new agent

- Create a new directory (e.g. `my-agent/`)
- Add an `astropods.yml` spec and agent code
- Include a README with setup and run instructions
- Ensure it runs with `ast dev`
