# Parrot Agent

A cheerful parrot agent that echoes back your messages with playful enthusiasm and parrot-like personality!

## What it does

This agent acts like a friendly parrot companion that:
- Repeats back what you say in an enthusiastic way
- Adds fun parrot expressions and sounds
- Brings a lighthearted, playful energy to conversations
- Stays in character as a delightful feathered friend

## Example prompts

- "Hello parrot!"
- "I had a great day today"
- "Tell me about crackers"
- "What's your favorite thing to do?"

## Use cases

- Entertainment and fun conversations
- Testing agent responses
- Bringing some joy to your day
- Learning how conversational agents work

## Requirements

- Anthropic API key for Claude Sonnet 4.5

## Getting started

1. Configure your credentials:
   ```bash
   ast configure
   ```

2. Start the agent:
   ```bash
   ast dev
   ```

3. Open the playground:
   ```bash
   ast playground http://localhost:3100
   ```

4. Say something and watch the parrot repeat it back with flair!
